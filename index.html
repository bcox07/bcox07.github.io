About Me

My name is Brian Cox and I am a Software Developer that will graduate from Georgia Southern University in May. I have an interest in back-end development, but I also enjoy learning anything involved with software development. Make sure to check out some of my recent projects or contact me here!

Portfolio

  • Theseus
    Theseus is a Survival/Puzzle game based on the Greek Myth know as Theseus and the Minotaur. It was designed using the Unity Game Engine paired with the C# programming language.
  • Updog
    Updog is a community driven music web application where the users create and add songs with the idea of collaboration. This is created using Sass and Pug for the front-end, Node.js with REST API for the back-end, and PostgreSQL for the database.
  • Osmoe Sam
    Osmoe Sam is a motivational web application that provides stats for the user's pet to keep the user motivated. This is created using Sass and Pug for the front-end, Node.js for the back-end, and PostgreSQL for the database.
  • Student Emmy
    I was recently awarded a Student Emmy for my work in Sports, Live Events - Georgia Southern Baseball.
